Create a complete React 19 + Vite + TypeScript + TailwindCSS project called "netco-vpn". This is a VPN config selling platform for Kenya. Generate ALL files listed below with complete, working code.
## PROJECT OVERVIEW
NETCO is a premium VPN internet config provider for Kenya. It sells device-locked VPN configurations for HTTP Custom and HTTP Injector apps. Users purchase plans for Safaricom, Airtel, or Telkom networks. Payments are processed via M-Pesa STK Push through a payment gateway called PayFlow (API at https://payflow.top/api/v2). The project has a dark cyberpunk theme with cyan (#00F5FF) and purple (#7B61FF) accents.
